# Pay My Buddy

Project realized with :
- SpringBoot 2.7.9
- SpringBoot Security
- MySQL 8.0.35
- Java 8
- Bootstrap v5.3.0

Pay My Buddy is an application who sets up and manages money transfer between users of the app.
